# eigvec

Eigenvectors of A


## Syntax

```
eigvec(A)
```

## Description

Eigenvectors of A

## Mathematical Formulation

$$ A v_i = \lambda_i v_i \quad\text{(columns are the eigenvectors)} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `A` | Number | Yes | Numeric argument. |
