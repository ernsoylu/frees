# trace

Matrix trace


## Syntax

```
trace(A)
```

## Description

Matrix trace

## Mathematical Formulation

$$ \operatorname{tr}(A) = \sum_i A_{ii} = \sum_i \lambda_i $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `A` | Number | Yes | Numeric argument. |
