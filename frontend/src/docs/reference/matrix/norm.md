# norm

Matrix norm


## Syntax

```
norm(A)
```

## Description

Matrix norm

## Mathematical Formulation

$$ \lVert v \rVert_2 = \sqrt{\textstyle\sum_i v_i^2} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `A` | Number | Yes | Numeric argument. |
