# stringval

Parse a numeric string literal to a value


## Syntax

```
StringVal(s$)
```

## Description

Parse a numeric string literal to a value

## Mathematical Formulation

$$ \operatorname{StringVal}(s) = \text{numeric value parsed from } s $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `s$` | String | Yes | String literal. |
