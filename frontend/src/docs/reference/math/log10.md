# log10

Base-10 logarithm


## Syntax

```
log10(x)
```

## Description

Base-10 logarithm

## Mathematical Formulation

$$ y = \log_{10}(x) = \frac{\ln x}{\ln 10}, \quad x > 0 $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
