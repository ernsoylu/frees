# sinh

Hyperbolic sine


## Syntax

```
sinh(x)
```

## Description

Hyperbolic sine

## Mathematical Formulation

$$ \sinh(x) = \frac{e^{x} - e^{-x}}{2} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
