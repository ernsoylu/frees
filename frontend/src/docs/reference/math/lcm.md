# lcm

Least common multiple


## Syntax

```
lcm(a, b)
```

## Description

Least common multiple

## Mathematical Formulation

$$ \operatorname{lcm}(a,b) = \frac{|a\,b|}{\gcd(a,b)} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `a` | Number | Yes | First operand. |
| `b` | Number | Yes | Second operand. |
