# arccos

Inverse cosine [rad] (alias of acos)


## Syntax

```
arccos(x)
```

## Description

Inverse cosine [rad] (alias of acos)

## Mathematical Formulation

$$ y = \arccos(x), \qquad \cos(y) = x,\ \ y \in [0, \pi] $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
