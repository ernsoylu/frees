# product

Product series Pi(term) over i = lo..hi


## Syntax

```
product(i, lo, hi, term)
```

## Description

Product series Pi(term) over i = lo..hi

## Mathematical Formulation

$$ \prod_{i=\text{lo}}^{\text{hi}} \text{term}(i) $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `i` | Number | Yes | Numeric argument. |
| `lo` | Number | Yes | Numeric argument. |
| `hi` | Number | Yes | Numeric argument. |
| `term` | Number | Yes | Numeric argument. |
