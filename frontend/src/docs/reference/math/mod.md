# mod

Modulo operation


## Syntax

```
mod(x, y)
```

## Description

Modulo operation

## Mathematical Formulation

$$ \operatorname{mod}(a,b) = a - b\,\lfloor a/b \rfloor $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
| `y` | Number | Yes | Value / second coordinate. |
