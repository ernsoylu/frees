# atan2

Four-quadrant inverse tangent


## Syntax

```
atan2(y, x)
```

## Description

Four-quadrant inverse tangent

## Mathematical Formulation

$$ \operatorname{atan2}(y,x) = \arg(x + jy) \in (-\pi, \pi] $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `y` | Number | Yes | Numeric argument. |
| `x` | Number | Yes | Numeric argument. |
