# arcsinh

Inverse hyperbolic sine


## Syntax

```
arcsinh(x)
```

## Description

Inverse hyperbolic sine

## Mathematical Formulation

$$ \operatorname{arcsinh}(x) = \ln\!\big(x + \sqrt{x^2+1}\big) $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
