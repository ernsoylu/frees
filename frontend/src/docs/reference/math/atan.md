# atan

Inverse tangent


## Syntax

```
atan(x)
```

## Description

Inverse tangent

## Mathematical Formulation

$$ y = \arctan(x), \qquad \tan(y) = x $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
