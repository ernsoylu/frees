# factorial

Factorial n!


## Syntax

```
factorial(n)
```

## Description

Factorial n!

## Mathematical Formulation

$$ n! = \prod_{k=1}^{n} k = n\,(n-1)! $$

$$ \quad n! = \Gamma(n+1) $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `n` | Number | Yes | Order / number of terms. |
