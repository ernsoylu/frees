# round

Round to nearest integer


## Syntax

```
round(x)
```

## Description

Round to nearest integer

## Mathematical Formulation

$$ \operatorname{round}(x, d) = \frac{\lfloor x\cdot 10^{d} + 0.5\rfloor}{10^{d}} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
