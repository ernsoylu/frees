# bitnot

Bitwise NOT


## Syntax

```
bitnot(a)
```

## Description

Bitwise NOT

## Mathematical Formulation

$$ (\sim a) = -(a+1)\ \text{(two’s complement)} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `a` | Number | Yes | First operand. |
