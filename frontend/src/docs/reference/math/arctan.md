# arctan

Inverse tangent [rad] (alias of atan)


## Syntax

```
arctan(x)
```

## Description

Inverse tangent [rad] (alias of atan)

## Mathematical Formulation

$$ y = \arctan(x), \qquad \tan(y) = x $$

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
