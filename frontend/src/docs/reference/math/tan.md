# tan

Tangent of x


## Syntax

```
tan(x)
```

## Description

Tangent of x

## Mathematical Formulation

$$ \tan(x) = \frac{\sin x}{\cos x}, \qquad x \text{ in radians} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
