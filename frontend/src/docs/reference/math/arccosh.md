# arccosh

Inverse hyperbolic cosine (x>=1)


## Syntax

```
arccosh(x)
```

## Description

Inverse hyperbolic cosine (x>=1)

## Mathematical Formulation

$$ \operatorname{arccosh}(x) = \ln\!\big(x + \sqrt{x^2-1}\big), \quad x \ge 1 $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
