# floor

Floor


## Syntax

```
floor(x)
```

## Description

Floor

## Mathematical Formulation

$$ \lfloor x \rfloor = \max\{n \in \mathbb{Z} : n \le x\} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
