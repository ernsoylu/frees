# tanh

Hyperbolic tangent


## Syntax

```
tanh(x)
```

## Description

Hyperbolic tangent

## Mathematical Formulation

$$ \tanh(x) = \frac{\sinh x}{\cosh x} = \frac{e^{x}-e^{-x}}{e^{x}+e^{-x}} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
