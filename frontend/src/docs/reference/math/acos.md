# acos

Inverse cosine


## Syntax

```
acos(x)
```

## Description

Inverse cosine

## Mathematical Formulation

$$ y = \arccos(x), \qquad \cos(y) = x,\ \ y \in [0, \pi] $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
