# abs

Absolute value


## Syntax

```
abs(x)
```

## Description

Absolute value

## Mathematical Formulation

$$ |x| = \begin{cases} x & x \ge 0 \\ -x & x < 0 \end{cases} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Vapor quality (0–1). |
