# bitor

Bitwise OR


## Syntax

```
bitor(a, b)
```

## Description

Bitwise OR

## Mathematical Formulation

$$ (a \mathbin{|} b)\ \text{— bitwise OR of the integer operands} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `a` | Number | Yes | First operand. |
| `b` | Number | Yes | Second operand. |
