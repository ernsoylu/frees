# Inverse

Matrix inverse A⁻¹


## Syntax

```
Inverse(A)
```

## Description

Matrix inverse A⁻¹

## Mathematical Formulation

$$ A\,A^{-1} = A^{-1}A = I $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `A` | Number | Yes | Matrix. |
