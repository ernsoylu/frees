# bitshiftr

Right bit shift a>>n


## Syntax

```
bitshiftr(a, n)
```

## Description

Right bit shift a>>n

## Mathematical Formulation

$$ a \gg n = \lfloor a / 2^{n} \rfloor $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `a` | Number | Yes | Numeric argument. |
| `n` | Number | Yes | Numeric argument. |
