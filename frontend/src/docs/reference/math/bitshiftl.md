# bitshiftl

Left bit shift a<<n


## Syntax

```
bitshiftl(a, n)
```

## Description

Left bit shift a<<n

## Mathematical Formulation

$$ a \ll n = a\cdot 2^{n} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `a` | Number | Yes | Numeric argument. |
| `n` | Number | Yes | Numeric argument. |
