# cosh

Hyperbolic cosine


## Syntax

```
cosh(x)
```

## Description

Hyperbolic cosine

## Mathematical Formulation

$$ \cosh(x) = \frac{e^{x} + e^{-x}}{2} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
