# sign

Sign function (-1, 0, or 1)


## Syntax

```
sign(x)
```

## Description

Sign function (-1, 0, or 1)

## Mathematical Formulation

$$ \operatorname{sign}(x) = \begin{cases} -1 & x<0 \\ 0 & x=0 \\ 1 & x>0 \end{cases} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
