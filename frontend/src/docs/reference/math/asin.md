# asin

Inverse sine


## Syntax

```
asin(x)
```

## Description

Inverse sine

## Mathematical Formulation

$$ y = \arcsin(x), \qquad \sin(y) = x,\ \ y \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}] $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
