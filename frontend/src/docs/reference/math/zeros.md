# zeros

m×n zero matrix


## Syntax

```
zeros(m, n)
```

## Description

m×n zero matrix

## Mathematical Formulation

$$ Z_{ij} = 0 \quad (m\times n) $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `m` | Number | Yes | Numeric argument. |
| `n` | Number | Yes | Numeric argument. |
