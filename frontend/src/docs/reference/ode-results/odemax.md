# odemax

Maximum of an ODE column


## Syntax

```
ODEMax('col')
```

## Description

Maximum of an ODE column

## Mathematical Formulation

$$ \max_i \text{col}(t_i) $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `'col'` | Number | Yes | Name of a result-table column (string). |
