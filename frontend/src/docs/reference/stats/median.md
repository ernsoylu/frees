# median

Median of vector x


## Syntax

```
median(x)
```

## Description

Median of vector x

## Mathematical Formulation

$$ \text{the middle order statistic (mean of the two middle values if } n \text{ even)} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
