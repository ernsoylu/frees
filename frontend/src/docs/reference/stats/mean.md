# mean

Mean of vector x


## Syntax

```
mean(x)
```

## Description

Mean of vector x

## Mathematical Formulation

$$ \bar x = \frac{1}{n}\sum_{i=1}^{n} x_i $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `x` | Number | Yes | Numeric argument. |
