# conj

Complex conjugate


## Syntax

```
conj(z)
```

## Description

Complex conjugate

## Mathematical Formulation

$$ \bar z = \overline{a + jb} = a - jb $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `z` | Number | Yes | Argument (complex or real). |
