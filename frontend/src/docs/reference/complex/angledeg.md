# angledeg

Argument of z [deg]


## Syntax

## Description

Argument of z [deg]

## Mathematical Formulation

$$ \arg(z) = \operatorname{atan2}(b, a)\cdot\tfrac{180}{\pi}\ \ [\text{deg}] $$
