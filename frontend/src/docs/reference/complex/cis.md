# cis

e^(j*theta) = cos(theta) + j*sin(theta)


## Syntax

```
cis(theta)
```

## Description

e^(j*theta) = cos(theta) + j*sin(theta)

## Mathematical Formulation

$$ \operatorname{cis}(\theta) = e^{j\theta} = \cos\theta + j\sin\theta $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `theta` | Number | Yes | Flow-deflection angle [rad]. |
