# machangle

Mach angle mu = asin(1/M) [rad]


## Syntax

```
MachAngle(M)
```

## Description

Mach angle mu = asin(1/M) [rad]

## Mathematical Formulation

$$ \mu = \arcsin\!\frac{1}{M} $$

## Input Arguments

| Argument | Type | Required | Description |
| --- | --- | --- | --- |
| `M` | Number | Yes | Numeric argument. |
